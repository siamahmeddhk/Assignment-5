Question-1: What are the different ways to select an element in the DOM?
There are five ways to select dom elements. they are :
1. getElementById("id")

2. getElementsByClassName("class")

3. getElementsByTagName("tag")

4. querySelectorAll("css-selector")

5. querySelectorAll("css-selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?
The difference between innerHTML, innerText & textContent is simple. Let me explain this
innerHTML : It give us whole html part. Can be used to insert HTML.
innerText : It gives only text which is written on inside html tags.
textContent : It also gives text but it gives all text..like visual and hidden all texts.
Question-3: What is event delegation in the DOM?
Event delegation is like when parent element handel events for its child element. It uses event bubble for this.
Question-4: What is event bubbling in the DOM?
Event bubbling means event triggered on a child element propagates to its parent and then to higher stage until it reaches the root
